# How to Play — Order Up!

## What Is Order Up!?

Order Up! is a collaborative puzzle game for **2 players** on the Board tabletop console. Work together to place physical game pieces in the correct order to solve logic puzzles — and learn the building blocks of coding along the way!

## Getting Started

1. **Gather 2 players** around the Board
2. The game launches into a **3-level tutorial** that walks you through the basics
3. After completing the tutorial, the **Level Select** screen opens and you can pick any unlocked puzzle

## How to Play

- **Read the task** displayed at the top of the screen (e.g., "Get ready for school!")
- **Pick up a game piece** and place it on the Board in one of the glowing slots
- The game tells you if your placement is correct:
  - **"Correct!"** (green) — Great, move on to the next slot!
  - **"Try again!"** (orange) — Pick up the piece and try a different one
- **Fill all the slots** in the right order to solve the puzzle
- **"You won! Great job!"** — Puzzle complete! You'll automatically move to the next challenge

**Tip:** Talk with your partner! Discuss which piece should go where before placing it.

## The Four Worlds

Each world teaches a different coding concept. Puzzles get more challenging as you progress.

### Enchanted Forest — Sequencing (Levels 1-5)

*Put steps in the right order.*

Place pieces in the correct sequence to complete a task. Example: to brew a potion, you need to gather herbs *before* you stir the cauldron!

- 3 tutorial levels to learn the basics
- 2 challenge levels to test your skills

### Castle Workshop — Procedures (Levels 6-8)

*Group steps together, then put the groups in order.*

Some steps belong together as a group. Complete one group before starting the next. Example: to forge a sword, first do all the "heating" steps, then all the "shaping" steps.

### Mystic Ocean — Loops (Levels 9-11)

*Spot the repeating pattern.*

Some tasks repeat the same steps over and over. Find the pattern and place it correctly each time. Example: rowing a boat is the same motion repeated — dip, pull, lift, dip, pull, lift!

### Dragon's Crossroads — Conditionals (Levels 12-14)

*Choose the right path based on a condition.*

The game gives you a condition (like "Is the dragon fire or ice?") and you pick the matching set of steps. The condition changes each time you play, so stay sharp!

## Unlocking Levels

| Complete... | To unlock... |
|---|---|
| Tutorial (Levels 1-3) | Enchanted Forest challenges (4-5) + Level Select |
| Level 4 | Castle Workshop (6-8) |
| Level 6 | Mystic Ocean (9-11) |
| Level 9 | Dragon's Crossroads (12-14) |

Locked levels appear dimmed on the Level Select screen. Completed levels show a gold checkmark.

## Tips for Players

- **Work together** — both players can move pieces at the same time
- **Read the hints** — if you're stuck, hints appear after incorrect placements
- **Try different orders** — there's no penalty for guessing wrong, just pick up the piece and try again
- **Watch the slots** — empty slots glow and pulse to show where pieces can go
- **Talk it out** — the best teams discuss their plan before placing pieces

## For Educators

Order Up! maps directly to four foundational programming concepts:

| World | Concept | What Students Learn |
|---|---|---|
| Enchanted Forest | **Sequencing** | Instructions must execute in a specific order |
| Castle Workshop | **Procedures** | Steps can be grouped into reusable blocks |
| Mystic Ocean | **Loops** | Patterns can repeat to reduce work |
| Dragon's Crossroads | **Conditionals** | Different conditions lead to different actions |

All interactions are logged to a CSV file at `[device]/interaction_logs/` for post-session analysis. The log captures which player placed which piece, where, and when — useful for understanding collaboration patterns and learning progression.
